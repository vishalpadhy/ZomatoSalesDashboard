# ZomatoSalesDashboard
This project showcases the power of data visualization and how DAX functions can unlock deeper insights into restaurant sales and customer preferences.

<p>𝐈𝐧𝐭𝐞𝐫𝐚𝐜𝐭𝐢𝐯𝐞 𝐕𝐢𝐬𝐮𝐚𝐥𝐬: Created dynamic visuals, including bar charts, pie charts, and city-specific insights, that make the data easy to explore and understand.</p>
